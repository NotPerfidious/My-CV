## Technologies Used
* **Frontend:** HTML5 and CSS3 (utilizing Flexbox for layout management)
* **Interactivity:** Vanilla JavaScript for DOM manipulation and theme logic
* **Typography:** Google Fonts (Montserrat for headings and Roboto for body text)
* **Deployment:** Vercel via GitHub Continuous Deployment

## Key Features
* **Responsive Layout:** Optimized for both mobile and desktop viewing using dynamic viewport units and Flexbox.
* **Dark/Light Mode (Bonus):** A custom theme switcher that dynamically updates background colors, container styles, and icon sets.
* **Downloadable CV (Bonus):** Integrated PDF download functionality for professional distribution.
* **Interactive Project Links:** Detailed technical breakdown and live repository links for major projects.
